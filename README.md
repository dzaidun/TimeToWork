Cтворено юнiт-тест, який перевіряє можливість створення та збереження клієнта в базі даних у твоєму веб-додатку "TimeToWork", використовуючи Entity Framework Core InMemory. Дозволяє перевірити, що об’єкт Client:
- створюється правильно,
- зберігається в базі,
- і потім успішно знаходиться через запит.
